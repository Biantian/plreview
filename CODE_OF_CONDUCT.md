# Code Of Conduct

## 我们希望的协作方式

PLReview 致力于建设一个友善、务实、尊重彼此时间的协作环境。无论你是首次提 issue、提交 PR，还是参与设计讨论，都希望你：

- 关注事实、上下文和可复现信息
- 尊重不同经验层级和沟通风格
- 对代码、文档和评审意见保持建设性
- 在分歧出现时优先讨论方案与证据，而不是针对个人

## 不可接受的行为

以下行为不适合出现在本项目社区中：

- 人身攻击、羞辱、威胁、骚扰或歧视性表达
- 恶意刷屏、钓鱼、引战或持续偏离主题
- 未经允许公开他人的私人信息
- 明知会破坏协作环境却反复进行的攻击性行为

## 维护者职责

维护者可以基于社区安全和讨论质量：

- 引导讨论回到问题本身
- 编辑、隐藏或删除明显不合适的内容
- 暂时或永久限制破坏性参与者的互动权限

## 反馈与处理

如果你遇到不当行为，请优先通过仓库维护渠道联系维护者并提供必要上下文。

如果仓库暂时没有私密联络方式，请开一个不含敏感内容的 issue，请求维护者转到私下沟通渠道处理。
